# Что
Пакет генерации Excel файлов из JSON файлов

Реализовывается поверх проекта:
https://github.com/Kerroline/Excel-From-Json

Доступные OS:
* MacOS (Darvin) x64
* Linux x64

# Зачем
Скорость генерации Excel файлов

# Как 
```cmd
composer require kerroline/gophp-excel
```
